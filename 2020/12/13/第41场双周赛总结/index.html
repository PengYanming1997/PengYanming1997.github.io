1685. 有序数组中差绝对值之和. Medium

利用数学规律求解 解题思路 : （1）如果题目中求的不是差的绝对值的和 而是差的绝对值的和 那么ans[i] = nums[i] * n - sum(n)； （2）由于题目中给出的数组是一个非递减数组 所以 nums[i]与其之前的元素的差是大于等于0的 而与其之后的元素的差是小于等于零的 而加上负数的绝对值相当于是减这个负数 （3）所以差的绝对值的和相当于可以写成 ans[i] = nums[i] * i - sum(i) - （nums[i] * n - sum(n) - nums[i] * i + sum(i)）

ans[i] = 2 * (nums[i] * i - sum(i)) + sum(n) - nums[i] * n 通过这个公式动态规划逐步求和 很容易就能得到结果

1686. 石子游戏 VI. Medium

贪心算法 解题思路 : Alice和Bob取石头的策略都是 尽可能取得自己这面分值最大的石头 而且这个石头在对方那面也尽可能大 如果这个石头在自己这面分值最大 但是在对方那面不是分值最大的 那么需要进行比较 如何比较呢？比较：每次选择的时候做比较 自己选择对方那面最大的会损失多少 以及 选择自己这面最大的对方会多得多少 两者之间做差值

其实也就是「每次都选择A[i] + B[i] 最大的那个元素」

理解：就比如Alice这面最大的是元素1 Bob那面最大的元素是2 现在轮到Alice做选择 她就要比较 A1 - A2 与 B2 - B1之间的大小 选择差值大的那个「差值越大自身获利越多」 也就是：如果（A1 - A2）- （B2 - B1）> 0 =》 （A1 + B1）- （A2 + B2）> 0 =》 选择元素1 =》 所以总结而言就是「每次都选择A[i] + B[i] 最大的那个元素」，最终比较两人手里的石子总价值
